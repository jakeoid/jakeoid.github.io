Name Checker

This isn't probably going to be a very popular tool but is just somethign I created because I hated the way those plethora of other PHP-monsterous machines were running. If I want to check if a usernames taken (or even the information behind it) I shouldn't have to reload the page and wait (on my slow internet anyway). Here is my alternative to that! Written completely in Javascript (of course). Probably not the most conservative thing ever but still usefull!

Also please note that this is very much still experimental.
