Romaji to Hiragana.

This is an extremely old resource, basically its transfers from Romaji (a, i, u, e, o) into ひらがな (あ, い, う, え, お). For what I used it for back in the day (which really wasn't much) it worked perfectly fine. Being written in April of 2015, it doesn't share the same level of modern application that i'd once would've liked so i've chosen to put it up here for the sake of having it up. It has been straight marked as to Legacy as again, if it broke I wouldn't care for to update it. Theres all too many of these out there and heck, this is just my own version of that.
