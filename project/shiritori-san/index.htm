Shiritori-san

A pretty cool resource that allows for you to play a game of Japanese Shiritori through the voice/text chatting application Discord. Designed for single server use in mind, this bot was on of my very first projects in Python and as of such the code is pretty poor, but it succeeds in getting the job done!

As this bot was designed for single server use there is no way that you can just hit deploy and have it begin running on your server, rather you will have to install it and host it yourself if you would like to make use. I've left some basic instructions inside of the source code on Github.
