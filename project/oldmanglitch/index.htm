Old Man Glitch.

This is a shortened way to generate what Pokémon you will receive if you make use of the Old Man Glitch. The Old Man is a Glitch that was in the Japanese and International releases of Pokémon Red & Blue. Inside of this glitch you could abuse the player's name when you first start the game in order to gain certain Pokémon when preforming a very specific set of steps.

Please make note that this also will only work with the Custom Pokémon trainer names (NEW NAME) and not the default trainers names, for example.. Red, Blue, Ash, Gary, etc. As they will have different Pokémon as to what this generator says, for more accurate information as to why this is, and to see which Pokemon you will recieve if this is your name please see Bulbapedia for more accurate information.
